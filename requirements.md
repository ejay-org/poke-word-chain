# 프로젝트: 포켓몬 음성 끝말잇기 (PokeVoice Hybrid)

## 1. 개요
사용자의 음성을 인식하여 포켓몬 이름으로 끝말잇기를 진행하는 웹 앱. 하이브리드 로직과 힌트 시스템을 통해 사용자 경험을 최적화함.

## 2. 하이브리드 로직 (Hybrid Flow)
1. **음성 인식**: 사용자의 음성을 텍스트로 변환.
2. **상태 판별**: 텍스트가 "힌트"를 요청하는 것인지, 단어를 말하는 것인지 판별.
3. **팩트 체크 (Local Logic)**: 
   - [단어 입력 시] `pokemonData.json` 기반 존재 여부, 끝말 규칙, 중복 여부 검증.
   - [힌트 요청 시] 규칙에 맞는 다음 단어 후보 중 하나를 추출하여 LLM에 전달.
4. **응답 생성 (LLM)**: 
   - 검증 결과 또는 힌트 후보를 바탕으로 포켓몬 마스터 페르소나 응답 생성.
   - 힌트 제공 시 단어 자체를 말하거나, "ㅇㅇ로 시작하는 포켓몬은 어때?" 식으로 유도.
5. **음성 출력**: 생성된 문장을 브라우저 음성으로 재생.

## 3. 기술 스택 (무료/오픈소스 기반)
- **Frontend**: React.js, Tailwind CSS, Vite
- **STT (음성 인식)**: Groq API (Whisper-large-v3)
- **LLM (대화/힌트 가이드)**: Google Gemini API (Gemini 2.0 Flash)
- **TTS (음성 출력)**: Web Speech API (SpeechSynthesis)
- **Data**: 로컬 `pokemonData.json`

## 4. 상세 규칙 및 기능
- **두음법칙**: '리' -> '이' 등 한국어 표준 두음법칙 허용.
- **중복 방지**: 사용된 단어는 재사용 불가.
- **힌트 시스템**: 
  - 사용자가 "힌트", "힌트 줘", "모르겠어" 등을 말하면 트리거.
  - 아직 사용되지 않은 정답 후보 중 하나를 선정하여 제공.

## 5. 포켓몬 도감 (Pokemon Encyclopedia)
- **도감 진입**: 메인 화면에서 '포켓몬 도감' 버튼을 통해 진입.
- **기능**: (추후 상세 기획 예정)
  - 포켓몬 목록 조회
  - 포켓몬 상세 정보 확인
